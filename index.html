🔌 Connect to Supabase

Enter your Supabase project credentials to connect the POS database. You only need to do this once — credentials are saved in your browser.

Where to find these:
1. Go to supabase.com/dashboard
2. Select your project → Settings → API
3. Copy Project URL and anon public key

⚠️ First-time setup: After connecting, click the "Run SQL Setup" button that appears to create all required tables automatically.
